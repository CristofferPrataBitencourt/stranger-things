Uma landing page desenvolvida para explorar animações modernas e interatividade. 
Este projeto foca na experiência do usuário através de transições suaves e um visual temático impactante.

## 🛠️ Tecnologias Utilizadas
* **GSAP (GreenSock Animation Platform)** - Animações de alta performance.
* **HTML5 & CSS3** - Estruturação e estilização avançada.
* **JavaScript** - Lógica de interatividade.
